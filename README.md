# SeniorConnect

A platform where students get paid help from "seniors" — senior students, employees,
or alumni — for:
- Resume building
- Mock interviews
- Tutoring / lectures
- Project collaborations
- Freelance opportunities

Open to students from any college, school, or university, looking for any kind of help.

## Stack
- Client: Vite + React + Tailwind
- Server: Node + Express + MongoDB (Atlas)
